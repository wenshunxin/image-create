# 智慧农业小程序

园区数字化综合管理平台 - 智慧农业小程序首页 HTML 实现

## 📱 项目简介

这是一个高度还原的智慧农业小程序首页 HTML 页面，采用移动端优先的响应式设计，包含完整的主题切换系统。

## ✨ 功能特性

### 核心功能
- 🏠 **首页**：农场数据概览、智慧监测、物联监控、植株管理
- 📋 **工单管理**：任务列表、状态筛选、工单详情、状态流转
- 📈 **数据统计**：产量统计、销售数据、设备在线率、能耗分析
- 🔔 **消息中心**：系统通知、告警提醒、任务通知
- ⚙️ **设置页面**：个人信息、主题切换、通知管理

### 主题系统
支持三种预设主题，点击切换立即生效：

| 主题 | 风格 | 主色调 |
|------|------|--------|
| 🌿 生态绿野 | 自然、生态 | `#28A32F` 农场绿 |
| 💻 科技智农 | 现代、科技 | `#1890FF` 科技蓝 |
| ☀️ 丰收暖阳 | 温暖、活力 | `#FA8C16` 丰收橙 |

### 交互特性
- 底部导航 Tab 切换
- 弹窗动画效果
- 点击背景关闭弹窗
- 实时时间更新
- 主题偏好本地存储

## 🚀 快速开始

### 方式一：直接打开
```bash
# 在浏览器中打开
open smart-farm.html
```

### 方式二：使用本地服务器
```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve .

# 访问 http://localhost:8080
```

## 🎨 主题切换

1. 点击底部导航「设置」
2. 选择「主题设置」
3. 点击喜欢的主题卡片

主题会自动保存到 localStorage，下次打开自动应用。

## 📱 页面截图

### 首页
- 农场选择器（支持多农场切换）

## 💻 技术栈

- **纯 HTML5 + CSS3 + JavaScript**
- CSS Custom Properties（主题变量）
- Flexbox & Grid 布局
- localStorage 持久化
- 零依赖，单文件
